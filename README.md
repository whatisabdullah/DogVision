# Dog Vision — Dog Breed Classification using Deep Learning

A computer vision project built with **TensorFlow** and **Python** for classifying dog breeds from images using deep learning and image processing techniques.

**Developer:** Muhammad Abdullah  
**University:** FAST-NU, Lahore — Spring 2026

---

## Tech Stack

| Layer | Technology |
|---|---|
| Language | Python |
| Framework | TensorFlow / Keras |
| Data Processing | NumPy + Pandas |
| Visualization | Matplotlib |
| Dataset Handling | SciPy |
| Environment | Jupyter Notebook / Google Colab |

---

## Features

- Dog breed image classification  
- Deep learning model training  
- Image preprocessing and augmentation  
- Model evaluation and prediction  
- Custom image testing support

## Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/dog-vision.git
cd dog-vision
```

### Install Dependencies

```bash
pip install tensorflow numpy pandas scipy matplotlib
```

---

## Running the Project

```bash
jupyter notebook Dog_vision.ipynb
```

Or run directly on Google Colab.

---

## Project Workflow

1. Load dataset and annotations  
2. Preprocess and resize images  
3. Train the deep learning model  
4. Evaluate model performance  
5. Predict dog breeds from images
